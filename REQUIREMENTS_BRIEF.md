# 要件定義 — 簡潔版

> 詳細は [README.md](./README.md) · 改定: [CONCEPT.md](./CONCEPT.md)

| 項目 | 内容 |
|------|------|
| **コンセプト** | **振り付けの自習用メモ**（動画の解釈は人間）。拍・カウント・メモ・アレンジを助け、3D で反復 |
| **ターゲット** | まず開発者本人の非公開検証。動画で独学する人のローカル利用 |
| **解決したい課題とその解決策** | **課題:** 動画だけでは形・拍が分かりにくい<br><br>**解決策:** 譜面 JSON + 拍同期 + **3D お手本** + ローカル処理 |
| **利用シーン** | 自宅で動画参照 + 3D お手本の区間ループ |
| **利用の流れ** | 動画 → 譜面生成 → **3D でカウント練習** |
| **類似既存アプリで表現** | 楽譜アプリ + 簡易 3D モーション参考 |
| **なぜあなたがこれを行うのか** | 独学の反復用に、形の目安を自分ですぐ作りたい |
| **最低限機能** | Phase 0: 譜面 JSON 生成・プレビュー。Phase 1: **3D アバター** + 練習 UI |
| **あったら良い機能** | 反転、細かい区間、JSON エクスポート（低優先） |
| **やらないこと・入れない機能** | 公開 PF、3 AI、SNS、再配信、実写モーキャプ、AR/VR、2D棒中心、カメラ比較（当面） |
| **後で決めること** | アバター素材、公開、AI、精度ライン、音源 |
