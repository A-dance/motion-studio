## Phase 0: 譜面 JSON の棒人間プレビュー（カウント送り）

### 目的

生成した譜面 JSON を、動画なしで **カウント送り** の棒人間として表示し、拍と形の対応を目視確認する。

### タスク

- [ ] `scripts/preview_score.py`（または簡易 HTML）を追加
- [ ] 譜面 JSON の `counts[]` を読み、棒人間を描画
- [ ] キー操作またはボタンで前後のカウントに移動
- [ ] `LOCAL.md` にプレビュー手順を追記

### 受け入れ条件

- `data/output/*_score.json` を開くと棒人間が表示される
- カウントを進めるとポーズが切り替わる
